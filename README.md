GlobalizeSpree
==============

Localizing model data in Spree using Globalize3.

Installation
============

Add to Gemfile.

Also, to be super-sure (I've ran into some trouble before that, needs investigation) add

gem "i18n"
gem "globalize3"

to your application Gemfile as well.
